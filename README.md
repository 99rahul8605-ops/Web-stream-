# Telegram Video Stream Bot

A Telegram bot that converts videos to streamable web links. Users can send videos to the bot and receive a link to stream the video online.

## Features
- Convert Telegram videos to web-streamable links
- Mobile-friendly video player
- View count tracking
- 7-day auto-cleanup of old videos
- Admin statistics
- User video management

## Deployment on Render (Free Plan)

### Prerequisites
1. Telegram Bot Token from [@BotFather](https://t.me/BotFather)
2. Render account (free tier)
